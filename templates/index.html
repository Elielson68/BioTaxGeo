SpecieGeo - Qualidade de dados

Esse software-web permite a análise de arquivos com dados referentes a ocorrência de Espécies. Os tipos de verificações realizadas por ele são: Análise das coordenadas, Verificação de erros de digitação, verificação de erros na localização.

Para utilizar o software basta enviar logo abaixo seu arquivo que deve ser verificado.
